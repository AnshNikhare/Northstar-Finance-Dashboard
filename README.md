# Northstar Finance Dashboard

A responsive finance dashboard built with React, Vite, TypeScript, and shared context-based state management.

## Features

- Overview dashboard with balance, income, and expense analytics
- Transaction management with filtering, sorting, and admin controls
- Role switching between Viewer and Admin
- Shared global search from the navbar across dashboard pages
- Insights and settings views with role-aware UI

## Run Locally

1. Install dependencies:

   npm install

2. Start development server:

   npm run dev

3. Create production build:

   npm run build

4. Preview production build:

   npm run preview
